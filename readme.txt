config file click.ini to set info: delay, location X Y, Mouseclick(1 = left click 2= right click), textinput. will exit at end of settings file

ini file click.ini( same location as exe)
[Settings0]
Delay=4
XPOS=250
YPOS=400
Mouseclick=1
Textinput=HelloWorld

[Settings1] 
Delay=1
XPOS=400
YPOS=400
Mouseclick=1
Textinput=HelloWorld
